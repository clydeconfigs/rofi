list of my rofi scripts

autobluetooth lists and connects to bluetooth devices
bm3 saves and shows lines text, like a real clipboard
calculator runs rofi-calc
dictionary a variety of dictionaries to search locally or online
dmenu runs rofi -dmenu
emoji lists emojis
exiti3 lists options like lock screen, reboot, etc.
filemanager runs rofi -filebrowser
genrandom generate random words, passphrases, strings, etc. of different length
google query a google search
ip show public ip and info
pass runs a modified version of rofi-pass
time shows the date and time
translate querys a text to translate in google translator
weblinks lists a list of web links
windowcd runs rofi -mode windowcd
